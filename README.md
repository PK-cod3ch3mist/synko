# How to use

### add

```bash
synko add [configname] [paths...]
```

### remove

```bash
synko remove [configname]
```

### index

```bash
synko index
```

### info

```bash
synko info [options]
```


# Install

Install synko with pip

```bash
  pip install synko
```
